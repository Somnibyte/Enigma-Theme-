# Installation 

1. Create a directory named `FontAndColorThemes` within `~/Library/Developer/Xcode/UserData/`:

  `mkdir -p ~/Library/Developer/Xcode/UserData/FontAndColorThemes/`
  
2. Place `Enigma.xccolortheme` file in `FontAndColorThemes`. 

3. Restart Xcode.
